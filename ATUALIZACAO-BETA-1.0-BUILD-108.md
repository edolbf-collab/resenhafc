# Beta 1.0 — Build 108

## Sorteio da espera

- Área dedicada dentro de cada evento futuro.
- Administrador e organizador selecionam individualmente quem participa.
- O sorteio funciona independentemente do limite máximo e da existência de excedentes.
- É possível definir quantos dos selecionados começarão na espera.
- Exibição instantânea ou revelação nome a nome.
- O resultado e a ordem da espera permanecem salvos.
- Um novo sorteio substitui integralmente a espera anterior.

## Churrasco

- Quando desativado, a tela mostra apenas o checkbox compacto para o administrador.
- Ao marcar o checkbox, o painel completo com valor e opções é aberto.
- Para os demais membros, a seção grande só aparece quando o churrasco está ativado.
