# Identidade visual — Resenha FC

## Marca escolhida
Emblema circular com jogadores, bola, confirmação e confraternização. A composição comunica futebol, organização do jogo e convivência entre amigos.

## Paleta
- Azul-marinho: `#071F4A`
- Azul de interface: `#0B356D`
- Verde: `#399737`
- Laranja de destaque: `#F08B19`
- Fundo claro: `#F5F7F8`

## Arquivos principais
- `brand/logo-resenha-fc.png`: marca completa para telas e materiais.
- `brand/brand-mark.png`: versão otimizada para cabeçalho.
- `icons/icon-192.png` e `icons/icon-512.png`: ícones PWA.
- `icons/maskable-icon-192.png` e `icons/maskable-icon-512.png`: ícones adaptativos Android.
- `icons/apple-touch-icon.png`: ícone de instalação no iPhone/iPad.
- `native-assets/android/`: pacote para futuro projeto Android nativo/Capacitor.
- `native-assets/ios/AppIcon.appiconset/`: catálogo pronto para Xcode.

## Regra de uso
Não distorcer, recolorir ou remover elementos do emblema. Em tamanhos muito pequenos, usar os arquivos de ícone já preparados, que possuem margem segura para os recortes do sistema.
